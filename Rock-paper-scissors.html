We can now play this game in a digital form with the computer, which was played under the bench at school by the teachers in our youth.

What is this?

This is a paper scissors rock game.

This game can read the players' patterns to determine the next steps for 'Computer' will take in order to win.

Buit with js and Bootstrap css framework.

How to play

  1. You can start the game by selecting "rock", "scissors" or "paper".
  2. Choose continuously until you create a pattern.
  3. Choose continuously until you create a pattern.
    • Rock, rock, rock, rock, etc.
    • Rock, scissors, rock, scissors, etc.
    • Rock, scissors, paper, rock, scissors, paper, etc
    • Paper, paper, paper, paper, etc
  4. See that you will lose.
